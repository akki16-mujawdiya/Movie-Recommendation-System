# 🎬 Movie Recommendation System

## 📌 Overview

A Machine Learning-based Movie Recommendation System that suggests similar movies using Content-Based Filtering and Cosine Similarity.

## 🚀 Features

- Movie Recommendation Engine
- Content-Based Filtering
- Cosine Similarity
- TMDB 5000 Dataset
- Machine Learning Project

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn

## 📂 Dataset

TMDB 5000 Movies Dataset

Files:
- movies.csv
- credits.csv

## ⚙️ Working

1. Load movies and credits datasets
2. Merge datasets
3. Extract genres, keywords, cast and director
4. Create tags column
5. Convert text into vectors using CountVectorizer
6. Calculate similarity using Cosine Similarity
7. Recommend top 5 similar movies
